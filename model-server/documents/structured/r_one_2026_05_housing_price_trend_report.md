# 2026년 5월 전국주택가격동향조사 보고서

- 문서 성격: monthly_housing_price_trend_report
- 원본 파일: `r_one_2026_05_housing_price_trend_report.pdf`
- 출처: R-ONE report seq 3928, file 4804
- 설명: 한국부동산원 R-ONE의 월간 전국 주택 가격 동향 리포트입니다.
## 핵심 요약

조사대상기간: 2026년 5월 1일 ~ 2026년 5월 31일.
공표일: 2026년 6월 15일.
전국 주택가격은 2026년 5월에 지난달 대비 상승했습니다.
전월대비 변동률은 전국 기준 매매가격지수 0.21%, 전세가격지수 0.35%, 월세통합가격지수 0.35%입니다.
수도권은 매매 0.46%, 전세 0.61%, 월세통합 0.56%로 전국보다 상승폭이 컸습니다.
지방은 매매 -0.02%, 전세 0.10%, 월세통합 0.16%로 매매는 소폭 하락했고 임대차 지표는 상승했습니다.
서울은 매매 0.90%, 전세 0.91%, 월세통합 0.81%로 상승폭이 컸습니다.
보고서 설명에 따르면 서울은 정주여건 양호 단지, 대단지, 역세권 중심으로 매수 및 임차 문의가 증가하며 상승했습니다.
