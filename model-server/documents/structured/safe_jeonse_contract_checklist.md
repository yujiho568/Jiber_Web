# 안심 전세계약 체크리스트

- 문서 성격: jeonse_contract_safety_checklist
- 원본 파일: `safe_jeonse_contract_checklist.pdf`
- 출처: https://www.rtech.or.kr/file/Safe_Jeonse_Contract_Checklist.pdf
- 설명: 전세계약 전·중·후 임차인이 확인해야 할 항목을 정리한 안내서입니다.
## 핵심 요약

활용 방법: 전세 계약 전 확인사항, 등기사항증명서, 선순위채권, 확정일자, 전입신고, 보증보험, 특약 질문에 우선 사용합니다.
핵심 항목: 전세 시세 조사, 권리관계 확인, 선순위채권 확인, 건축물대장 확인, 소유자와 계약자 일치 여부 확인, 계약 후 신고와 확정일자, 잔금 지급 후 전입신고와 전세보증보험 가입.
